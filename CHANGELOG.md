# JadeDbClient Changelog

## [1.5.9] - 2026-03-03

### Added
- **QueryBuilder.CountAsync()**: Support for executing `SELECT COUNT(*)` queries asynchronously while respecting fluent `Where` filters and `Join` clauses.
- **Explicit Auto-Increment Support**: Properties can now be marked with `[JadeDbColumn(IsAutoIncrement = true)]` to be automatically excluded from `INSERT` and `UPDATE` statements generated by `QueryBuilder`.
- **Custom Mapping for Auto-Increment**: `JadeDbColumnAttribute` now supports a parameterless constructor allowing it to be used purely for marking auto-increment status without renaming the column.

### Changed
- **Insert/Update Behavior**: `QueryBuilder` no longer uses a name-based convention (like "Id") to skip columns. It now relies strictly on the `IsAutoIncrement` property of `JadeDbColumnAttribute` to ensure maximum flexibility for tables without standard ID fields.
- **Source Generator Update**: The AOT-safe source generator now respects the `IsAutoIncrement` attribute and excludes marked properties from bulk insert accessors.
